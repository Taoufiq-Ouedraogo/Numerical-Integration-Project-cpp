Detailed Description

Transform a 1D integral using x = t^2 substitution.

This changes int_{0}^{1} f(x) dx into int_{0}^{1} f(t^{2}) * 2t dt. Useful for functions not defined at x=0 (e.g., log(x), 1/sqrt(x)).
