Detailed Description

Monte Carlo integration using uniform sampling.

Generates random points uniformly in the interval and estimates the integral using the average value of the function.

Parameters

IntegrationResult with value, error estimate, etc.

Approximates: [ \int_a^b f(x)\,dx \approx (b-a)\,\frac{1}{n}\sum_{i=1}^n f(X_i), \quad X_i \sim \mathcal{U}(a,b) ]

Uses Welford's algorithm for numerically stable mean and variance estimation.

Standard error: [ \sigma_{\hat I} = (b-a)\sqrt{\frac{\mathrm{Var}(f)}{n}} ]
